How to set DVR Recording automatically:

1 Select a keyword using the user prefence screen. It can be any thing like artist name, director, or any string
2 Maximum 20 keyword can be selected, and maximum 20 programs can be recorded (based on storage spaces availability)
3 Priority would be given to highly rated movies
4 Batch process runs everyday to record the programs based on selected keywords. Keywords can be from title of the movie, artist names, cast etc.
5 Customer can view the currently recorded and scheduled programs through Recording status screen.
6 Customer can delete the movies recorded as well as scheduled
7This way customer will be able to watch the program of their favorite stars, without manually schedule it for recording
